Conversor de unidades V1 - 26/06/2026.

# Objetivos principais:
- O usuário deve ser capaz de converter entre unidades de medidas simples.
- O usuário deve ser capaz de converter entre diferentes moedas.

## 1. Principais unidades de medidas:
- **Sistema Internacional de Unidades**: comprimento, massa, temperatura e tempo.

## 2. Principais moedas:
- Dólares
- Reais
- Euros
- Renmimbi (Yuan)

### Principais funcionalidades:
1. Terminal que se limpa a cada operação concluída.
2. Confirmação de operação.
3. tela de descanso antes de cada operação.
4. Configuração de moedas manualmente
